Welcome to PiwikAnalyticsJs a PrestaShop module

Piwik Traking module for Prestashop (1.4,1.5,1.6).
A simple little module to track your customers in your online store build with Prestashop, track pages, carts, orders, product and category views, purchased products, total order amount, tax amount, shipping etc. etc. all the standard reports that are important to an online store

Piwik is an Open Source Website/Ecommerce Analytics software similar to "Google Analytics" but you are in complete control over the data

This module uses only JavaScript to track your customers, the JavaScript code is build by the module and appended to your store in the footer,

Don't hesitate to post issues or general questions, it the only way a project gets better
